## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) >= 1.0.0

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mhodngan-kmutt/mhodngan-kmutt-backend.git
   cd Mhodngan-backend
   ```

2. **Install dependencies**
   ```bash
   bun install
   ```

3. **Run development server**
   ```bash
   bun run dev
   ```

4. The server will start at [http://localhost:3000](http://localhost:8080)
